## 方法论说明

本分析基于 CaseFlow 平台的 **GraphRAG 案例推荐引擎**：

1. **概念层映射**：每个案例的微观实体（行动者、政策、空间干预等）通过关系连接到跨案例的抽象概念层（Schema 3 中的 case_id = 0）
2. **Jaccard 相似度**：计算两个案例的概念邻居集合的交集/并集比值
3. **三路融合**：综合图结构相似度（0.5）+ 案例向量余弦相似度（0.3）+ 实体类型分布相似度（0.2）
4. **深度解释**：AI 基于检索到的共享概念和子图结构，自动生成对比分析和决策建议
